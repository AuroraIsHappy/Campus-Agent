# 杭州旅游信息汇总

## 主要景点信息

### 1. 西湖风景区
- **位置**: 杭州市西湖区
- **开放时间**: 全天开放
- **门票价格**: 免费
- **当前开放状态**: 开放
- **近期活动**: 
  - 西湖音乐喷泉（每晚19:30、20:30各一场）
  - 断桥残雪观景（冬季特色）
- **备注**: 杭州核心景点，免费开放

### 2. 灵隐寺
- **位置**: 杭州市西湖区灵隐路法云弄1号
- **开放时间**: 07:00-18:15（夏令时），07:00-17:15（冬令时）
- **门票价格**: 
  - 飞来峰景区：45元
  - 灵隐寺：30元（需先购飞来峰门票）
- **当前开放状态**: 开放
- **近期活动**: 
  - 新年祈福活动（春节期间）
  - 佛事法会（定期举办）
- **备注**: 杭州著名古刹，建议早上前往

### 3. 西溪湿地
- **位置**: 杭州市西湖区天目山路518号
- **开放时间**: 07:30-18:30（夏令时），08:00-17:30（冬令时）
- **门票价格**: 
  - 周一至周四：80元
  - 周五至周日及节假日：100元
- **当前开放状态**: 开放
- **近期活动**: 
  - 西溪龙舟节（端午节期间）
  - 芦花节（秋季）
  - 听芦节（冬季）
- **备注**: 建议预留半天时间

### 4. 宋城
- **位置**: 杭州市西湖区之江路148号
- **开放时间**: 10:00-21:00
- **门票价格**:

3. **龙井虾仁** - 清淡爽口，茶香四溢
4. **叫化鸡** - 传统名菜，香酥可口

## 杭州市内交通费用

### 地铁
- **起步价**: 2元（4公里以内）
- **计价规则**: 
  - 4-12公里：3元
  - 12-22公里：4元
  - 22-32公里：5元
  - 32公里以上：每10公里加1元
- **运营时间**: 约06:00-23:00
- **主要线路**: 
  - 1号线：湘湖-下沙江滨
  - 2号线：朝阳-良渚
  - 4号线：浦沿-彭埠
  - 5号线：金星-姑娘桥
- **支付方式**: 支付宝/微信乘车码、公交卡、现金购票

### 公交
- **起步价**: 2元
- **计价规则**: 普通公交2元，空调公交2-4元
- **运营时间**: 约06:00-22:00（夜班车至23:30）
- **特色公交**: 
  - Y字头旅游专线（西湖周边）
  - 环湖观光巴士
- **支付方式**: 支付宝/微信乘车码、公交卡、现金

### 打车/网约车
- **起步价**: 
  - 出租车：13元/3公里
  - 网约车（滴滴）：根据车型和时段浮动
- **计价规则**:
  - 出租车：超过3公里后2.5元/公里
  - 网约车：动态定价，通常1.5-3元/公里
- **典型区段价格参考**:
  - 火车东站-西湖：约25-35元
  - 西湖-灵隐寺：约20-30元
  - 西湖-宋城：约40-50元
  - 灵隐寺-西溪湿地：约25-35元
  - 市区内短途：15-25元
- **高峰时段**: 工作日07:00-09:00、17:00-19:00价格上浮

### 共享单车
- **费用**: 约1.5元/30分钟
- **品牌**: 哈啰、美团、青桔
- **适用场景**: 短途接驳、西湖骑行

### 其他交通建议
- **西湖景区**: 建议步行或骑行，部分区域有观光车
- **景区接驳**: 多个景区间可利用地铁+公交/打车组合
- **旅游专线**: Y1-Y9路旅游公交连接主要景点

## 实用贴士

### 最佳游览时间
- **春季**（3-5月）：气候宜人，花季盛开
- **秋季**（9-11月）：天高气爽，桂花飘香
- **避开**：节假日和周末高峰期

### 购票建议
- 提前网上预订热门景点门票（宋城、灵隐寺）
- 关注官方公众号获取最新优惠信息
- 学生证、老年证可享受优惠票价

### 住宿建议
- **西湖周边**: 价格较高，交通便利
- **武林广场**: 商业中心，美食丰富
- **火车东站附近**: 交通枢纽，性价比较高

### 注意事项
- 西湖景区节假日限行，建议公共交通
- 寺庙参观注意着装要求
- 保管好个人财物，人多处注意安全
